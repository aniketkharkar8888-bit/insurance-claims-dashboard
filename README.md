# insurance-claims-dashboard
Power BI dashboard analyzing insurance claims data with KPIs, trends, and interactive filters for business insights.
Insurance Claims Analysis Dashboard
📊 Project Overview

This project analyzes insurance claims data using Power BI to generate business insights.

🔍 Key Metrics
Total Claims: 200
Total Claim Amount: ₹21M
Approval Rate: 33.5%
Avg Settlement Time: 27 days
📌 Features
Claims by Status (Approved, Pending, Rejected)
Monthly Claim Trends
Policy Type Distribution
Interactive Filters (City, Policy Type, Status)
🛠 Tools Used
Power BI
Excel
DAX
